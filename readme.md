# Graphitti

## Setup

### 1. Clone Repository

```bash
git clone <repo-url>
cd Graphitti
```

### 2. Start Neo4j

```bash
docker compose up -d
```

### 3. Install Dependencies

```bash
pip install -r requirements.txt
```

### 4. Load Graph Data

```bash
py connect.py
```

### 5. Open Neo4j Browser

http://localhost:7474

Username: neo4j

Password: password
